# Weather Clear App

Aplicación para revisar el pronóstico del clima usando la estética de Minecraft!

## Funcionalidades: 
- Consultar el clima de cualquier ciudad de Chile!
- Weather App cuenta con un widget minimalista que puedes agregar a tu menú, tú eliges la ciudad que se mostrará!
- Tienes la opción de usar tu ubicación actual para consultar el clima de ese preciso lugar, haciendo uso de las coordenadas de ubicación.
- Agregar ciudades favoritas a la pantalla principal en la sección de Búsquedas Rápidas para consultar el clima más rápido.
- La aplicación está disponible en dos idiomas: Inglés y Español. Se adapta al idioma de tu celular.
